# PRD: Professional Portfolio & Research Hub

## Project Overview
A high-authority, Jekyll-based portfolio designed to showcase the engineering expertise of Chhayansh Porwal. The site serves as a bridge between high-level Android/AI projects and the Japanese tech market.

## Core Objectives
1. **Showcase Technical Depth:** Move beyond "what it is" to "how it was built" (Problem/Solution/Tech Stack).
2. **Academic Integration:** Host research papers and competition summaries (e.g., AI4Life).
3. **Monetization:** Clean integration of Google AdSense without compromising UX.
4. **Professional Branding:** Align with the standards of top-tier Japanese tech firms.

## Information Architecture
*   **Home:** High-level bio, tech stack, and Japanese language goals.
*   **Experience:** Chronological breakdown of internships (Syon, Fotographiya).
*   **Portfolio:** Detailed case studies for CallSentry, Project Koma, NEP Cutter, etc.
*   **Publications:** Research-heavy section for AI4Life and communication control systems.
*   **Blog:** Insights on tech (Kotlin/FastAPI) and book reviews.

## Technical Requirements
*   **Framework:** Jekyll (AcademicPages template).
*   **Styling:** Clean, minimalist, responsive.
*   **Visuals:** Where screenshots are missing, use Mermaid.js syntax to create logic flowcharts.
*   **SEO:** Metadata for every project and blog post.
